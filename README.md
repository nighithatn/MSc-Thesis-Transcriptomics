# Data Availability

Raw RNA-seq FASTQ files used in this study were obtained from the NCBI Sequence Read Archive (SRA).

SRA Accession: SRP432690  
Samples: 10 Inflammatory Aortic Aneurysm (IAA), 10 Noninflammatory Aortic Aneurysm (NIAA)

Due to large file size, raw sequencing data are not included in this repository.
